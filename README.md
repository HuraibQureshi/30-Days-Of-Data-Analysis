# 📊 30 Days of Data Analysis Challenge

Welcome to my daily learning repository! This project documents my journey to mastering Data Analysis from scratch. Every single day, I dedicate time to learning a new concept, writing robust code, processing datasets, and building visual models.

## 🎯 Project Objectives
* Master advanced data manipulation engines (Pandas, NumPy).
* Implement relational database models and analytical queries using SQL.
* Build interactive business intelligence dashboards (Tableau, Plotly).
* Apply quantitative statistical modeling and risk tracking.
